## SimpleIcons Color Customization

SimpleIcons CDN supports custom colors via URL parameters. Examples:

- Default: `https://cdn.simpleicons.org/python`
- Custom color: `https://cdn.simpleicons.org/python/3776AB`
- Black: `https://cdn.simpleicons.org/linux/000000`
- Dark gray: `https://cdn.simpleicons.org/swagger/222222`

## Update Protocol

When adding new logos in the future:

1. **Search for SimpleIcons version first**: `https://simpleicons.org/`
2. **Check official repositories**: GitHub, etc.
3. **Use Wikimedia Commons**: For official brand logos
4. **Maintain consistency**: Use the same source types (CDN preferred)

## Backup Strategy

All light mode URLs are from reliable sources:

- ✅ **SimpleIcons CDN** - Fast, reliable, always available
- ✅ **Wikimedia Commons** - Permanent, official logos
- ✅ **GitHub Raw** - Official repositories
- ✅ **Google Static** - Google's official CDN
